## ðŸŽ¯ How to Launch

### Step 1: Environment Setup (15 minutes)
1. Create Supabase account at supabase.com
2. Copy Project URL and Anon Key
3. Create `.env.local` with credentials
4. Run `npm install` and `npm run dev`
5. Test locally at http://localhost:3000

### Step 2: Database Setup (5 minutes)
1. Copy SQL from `supabase/migrations/001_initial_schema.sql`
2. Paste in Supabase SQL Editor
3. Execute to create all tables with RLS

### Step 3: Admin Account (5 minutes)
1. Create admin user in Supabase Auth
2. Add to `admin_users` table
3. Test login at /admin/login

### Step 4: Production Build (2 minutes)
1. Run `npm run build`
2. This creates `/dist` folder

### Step 5: Deploy to Hosting (varies)
1. Upload `/dist` contents to hosting
2. Upload `.htaccess` (Apache) or configure nginx
3. Point domain to hosting
4. Set up HTTPS/SSL
5. Done! ðŸŽ‰

**Total time to live: ~30 minutes**

## ðŸ“Š Database Schema

### Products Table
```sql
id, name, description, price, category, 
images (array), weight_kg, is_active, created_at
```

### Orders Table
```sql
id, user_email, status, subtotal, shipping_cost, 
total, pudo_size, shipping_address (JSON), created_at
```

### Order Items Table
```sql
id, order_id (FK), product_id (FK), 
quantity, price, created_at
```

### Admin Users Table
```sql
id (FK to auth.users), email, role, created_at
```

All with proper indexes and Row Level Security policies.
